# Установка
Создание пользователей
```
  php yii create-users
```
# Перед началом создайте пользователей
Потом запустите команды для заполнения бд данными:
Создание занятий
```
  php yii sessions
```
Создание привязка пользователя к занятиям
```
  php yii user-sessions
```
Создание файлов
```
  php yii files
```
Создание логов
```
  php yii logs
```
